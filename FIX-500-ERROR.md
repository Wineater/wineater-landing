# Исправление 500 ошибки для Google Search Console

## 🚨 Проблема
Google Search Console показывает 500 ошибку сервера для wineater.com, что препятствует индексации сайта.

## ✅ Выполненные исправления

### 1. Упрощение конфигурации Nuxt
- Удалены сложные JSON.stringify в SSR контексте
- Упрощены модули SEO
- Убраны потенциально конфликтующие настройки

### 2. Исправление robots.txt
- Удалена неподдерживаемая директива `Crawl-delay`
- Упрощена структура файла
- Убраны блокировки важных путей

### 3. Упрощение SEO плагина
- Убраны сложные функции, которые могут вызывать ошибки
- Оставлена только базовая функциональность
- Все операции выполняются только на клиенте

### 4. Удаление проблемного middleware
- Удален глобальный SEO middleware
- Убраны сложные функции, которые могут вызывать ошибки сервера

## 🔧 Что было изменено

### Файлы изменены:
- `nuxt.config.ts` - упрощена конфигурация
- `public/robots.txt` - исправлен формат
- `plugins/seo.client.js` - упрощен плагин
- Удален `middleware/seo.global.ts`

### Файлы удалены:
- `middleware/seo.global.ts` - вызывал ошибки сервера
